ğŸš€ Getting Started

Prerequisites

Node.js (v18.x or later)

npm / yarn

Installation & Setup

Clone the repository:

git clone [https://github.com/supernexsys/hbr_analytics.git](https://github.com/supernexsys/hbr_analytics.git)
cd hbr_analytics


Install Server Dependencies:
(Assuming backend code is in a /server directory)

cd server
npm install


Install Client Dependencies:
(Assuming frontend code is in a /client directory)

cd ../client
npm install


Run the Application:

Run the server: (From the /server directory)

npm run dev


Run the client: (From the /client directory)

npm run start
